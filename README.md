## О проекте

Программный комплекс для прогнозирования дневной реализованной волатильности
(Realized Volatility) золота (`GLDRUB_TOM`) и серебра (`SLVRUB_TOM`) на
Московской бирже. Помимо классической эконометрической модели HAR-RV, в прогноз
включается многоисточниковый вектор настроений (sentiment) и внимания (attention)
инвесторов, собираемый из новостей, Telegram, поисковых запросов и
макроэкономических данных.

Проект охватывает весь цикл: сбор данных из семи источников → расчёт
реализованной волатильности → обучение и сравнение четырёх моделей →
веб-визуализация результатов.

## Основные результаты

Оценка методом расширяющегося окна (expanding window OOS), метрики унифицированы
(Patton QLIKE), sentiment-признаки лагируются на 1 день во избежание утечки данных.

| Инструмент | Модель | MAE | RMSE | R²_OOS | QLIKE |
|---|---|---|---|---|---|
| Золото | HAR-log | 0.674 | 0.869 | 0.177 | 0.628 |
| Золото | **HAR-log-S** | **0.672** | **0.868** | **0.180** | **0.626** |
| Золото | XGBoost | 0.725 | 0.924 | 0.070 | 0.627 |
| Золото | kNN (k=75) | 0.671 | 0.877 | 0.163 | 0.683 |
| Серебро | **HAR-log** | **0.838** | **1.060** | **0.377** | **0.979** |
| Серебро | HAR-log-S | 0.834 | 1.060 | 0.376 | 1.022 |
| Серебро | XGBoost | 0.879 | 1.119 | 0.305 | 1.237 |
| Серебро | kNN (k=30) | 0.859 | 1.086 | 0.345 | 1.167 |

**Вывод:** добавление sentiment даёт небольшой и статистически слабо значимый
выигрыш для золота (HAR-log-S — лучшая модель); для серебра лучший результат у
классической HAR-log. Более сложные XGBoost и kNN не превосходят HAR ни на одном
инструменте. Проверка значимости — тестом Диболда–Мариано.

## Источники данных

| Источник | Что даёт | Модуль |
|---|---|---|
| MOEX ISS API | OHLCV-котировки металлов | `src/data/download_moex.py` |
| RSS (6 СМИ) | новостной sentiment | `src/data/parse_rss.py` |
| Telegram (12 каналов) | sentiment розничных инвесторов | `src/data/parse_telegram.py` |
| GDELT GKG | тональность мирового новостного потока | `src/data/parse_gdelt.py` |
| Google Trends | индекс внимания (attention) | `src/data/parse_google_trends.py` |
| ЦБ РФ (XML API) | курсы валют, ключевая ставка | `src/data/parse_cbr.py` |
| FRED | Brent, VIX, DXY, CPI, ставка ФРС | `src/data/parse_fred.py` |

Тональность текстов извлекается моделями BERT (`blanchefort/rubert-base-cased-sentiment`
для русского, `nlptown/bert-base-multilingual` для английского).

## Запуск

```bash
# 1. зависимости
pip install -r requirements.txt

# 2. (опционально) обновить котировки с MOEX
bash refresh_moex.sh

# 3. запустить веб-дашборд
python3 app.py        # → http://127.0.0.1:5000
```

Дашборд состоит из трёх страниц: обзор волатильности, прогноз модели и
sentiment-монитор.

## Воспроизведение моделей

```bash
python3 -m src.models.compute_rv      # реализованная волатильность (Garman–Klass)
python3 -m src.models.har_model       # HAR-log и HAR-log-S
python3 -m src.models.knn_model       # kNN на пространстве информационных состояний
python3 -m src.models.ml_models       # XGBoost (требует пакет xgboost)
```

## Технологии

Python 3.11, pandas, numpy, scikit-learn, xgboost, transformers (BERT),
Flask, Plotly, статья оформлена в LaTeX.

---

*Реализованная волатильность считается оценщиком Garman–Klass по дневным
OHLC-ценам; внутридневные тиковые данные по инструментам MOEX недоступны.*
